Hello to Crystal Reports and PowerBuilder fans.

This ZIP file contains a sample PowerBuilder application that contains 
the most commonly used Crystal Reports Print Engine functions.  The 
Print Engine (PE) functions have been declared under the global external 
functions.  With all the different PE calls which have been declared and 
used in this applications you should be able to implement any of the 
remaining PE call.

Please feel free to copy and use any code from this application.
